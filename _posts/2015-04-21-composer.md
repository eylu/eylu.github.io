## 安装 composer

### windows 安装方法

1、下载 - [Composer](https://getcomposer.org/Composer-Setup.exe)
2、安装
3、测试

```
> composer
```

### linux 安装方法

1、执行命令，php其实是php安装目录中bin目录里的php可执行文件路径

```
curl -sS https://getcomposer.org/installer | php
curl -sS https://getcomposer.org/installer | /pathto/php/bin/php
```
